The .table-responsive class creates a responsive table which will scroll horizontally on small devices (under 768px). When viewing on anything larger than 768px wide, there is no difference:

classname description contextual class
.table light padding and only horizontal dividers active
.table-striped zebra stripes success
.table-bordered border for entire table and also all cells info
.table-hover add hover effects on rows - gray colour warning
.table-condensed compact table with cellpadding as half danger
